PNEUMONIA DETECTION AI APP
=============================

Cara Install dan Jalankan:

1. Download seluruh folder ini ke komputer
2. Pastikan Python sudah terinstall (python.org)
3. Buka CMD/Terminal di folder ini
4. Untuk Windows: Double klik run.bat
5. Untuk Mac/Linux: Jalankan ./run.sh
6. Buka http://localhost:5000 di browser

File Structure:
- code + dataset/    -> code model & dataset, pipeline work model
- model/             -> model_EFFICIENTNET_final.keras -> AI Model
- output             -> paper
- sample_data_test   -> image for test the project
- templates/          -> HTML templates 
- app.py              -> Main application
- requirements.txt    -> Python dependencies 
- run.bat            -> Windows runner



System Requirements:
- Python 3.8+
- 4GB RAM minimum
- Windows/Mac/Linux

Note: web ini untuk bantuan diagnosis, konsultasi dokter tetap diperlukan.
